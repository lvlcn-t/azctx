## azctx list

List all available Azure contexts

### Synopsis

List all available contexts from the merged azctx config.

```
azctx list [flags]
```

### Examples

```
  azctx list
  azctx list -o table
  azctx list -o json
```

### Options

```
  -h, --help            help for list
  -o, --output string   Output format. One of: text|table|json (default "text")
```
