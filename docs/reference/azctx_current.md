## azctx current

Show the current active context

### Synopsis

Show the current active context from azctx config.

```
azctx current [flags]
```

### Examples

```
  azctx current
  azctx current -o json
  azctx current --verbose -o table
```

### Options

```
  -h, --help            help for current
  -o, --output string   Output format. One of: text|table|json (default "text")
  -v, --verbose         Include full context details
```
