## azctx rename-context

Rename a context in azctx config

### Synopsis

Rename a context in azctx config.

```
azctx rename-context OLD_NAME NEW_NAME [flags]
```

### Examples

```
  azctx rename-context staging dev
```

### Options

```
  -h, --help   help for rename-context
```
