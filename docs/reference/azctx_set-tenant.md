## azctx set-tenant

Set a tenant entry in azctx config

### Synopsis

Set a tenant entry in azctx config.

```
azctx set-tenant NAME [flags]
```

### Examples

```
  azctx set-tenant corp --id 00000000-0000-0000-0000-000000000000
```

### Options

```
  -h, --help        help for set-tenant
      --id string   Tenant ID
```
