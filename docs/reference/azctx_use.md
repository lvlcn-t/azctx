## azctx use

Set the active Azure context

### Synopsis

Set the active Azure context, then sync Azure CLI state by calling az login and az account set.

```
azctx use NAME [flags]
```

### Examples

```
  azctx use dev
  azctx use prod
```

### Options

```
  -h, --help   help for use
```
