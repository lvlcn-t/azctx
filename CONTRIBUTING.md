# Contributing

Contributions are welcome. This document covers the local development
workflow so you can get up and running quickly.

## Prerequisites

- Go 1.26+
- `GOEXPERIMENT=jsonv2` — **must** be set for build, test, vet, and
  lint (the Makefile exports it, but set it in your shell if running
  commands directly)
- [pre-commit](https://pre-commit.com/) (recommended)
- [gofumpt](https://github.com/mvdan/gofumpt) (formatting)
- [golangci-lint](https://golangci-lint.run/) (linting)

## Build and test

```bash
make build           # binary → bin/azctx
make dev use         # build + run a subcommand
make test            # go test -race -cover ./...
```

## Pre-commit hooks

Install hooks once, then they run automatically on every commit:

```bash
pre-commit install
```

To run all hooks manually:

```bash
pre-commit run --all-files
```

Hooks execute in order:

1. `go generate ./...` — regenerates mocks
2. `go mod tidy`
3. `go test -race -count=1 -short`
4. `go vet`
5. `gofumpt -l -w`
6. `golangci-lint --fix`
7. `gitleaks` — secret scanning

## Code generation

Mock files (`*_moq.go`) are generated by `go tool moq`. After changing
an interface in `az/`, `keyvault/`, or `wif/`:

```bash
GOEXPERIMENT=jsonv2 go generate ./...
```

Never hand-edit `*_moq.go` files — they are overwritten on every
generate run.

## Pull request expectations

- All pre-commit hooks pass (`pre-commit run --all-files`)
- Tests pass with `-race` and `-short`
- No `//nolint` without a specific linter name
- Commit messages are concise and describe the "why"

## Project structure

| Path        | Purpose                                    |
| ----------- | ------------------------------------------ |
| `cmd/`      | Cobra command definitions                  |
| `az/`       | Azure CLI integration                      |
| `config/`   | Config loader, writer, types               |
| `keyvault/` | Key Vault secret retrieval                 |
| `wif/`      | Workload identity federation               |
| `output/`   | Output formatting                          |
